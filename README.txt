# Currency Converter

A simple web-based currency converter built using HTML, CSS, and JavaScript.

## Features

- User inputs amount
- Selects "From Currency" and "To Currency"
- Calculates conversion using mock exchange rates
- Displays converted amount
- Clean user interface with responsive design

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Next Steps (Future Expansion)

- Connect to real-time exchange rate API
- Add more currencies
- Improve error handling
- Deploy live via GitHub Pages
